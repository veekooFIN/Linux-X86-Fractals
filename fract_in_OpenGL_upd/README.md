Some tips

How to install OpenGL/freeglut:

sudo apt update

sudo apt install freeglut3-dev libgl1-mesa-dev libglu1-mesa-dev

How to compile:

gcc gfxbrot.c -o gfxbrot -lglut -lGL -lGLU

How to run:

From terminal ./gfxbrot
