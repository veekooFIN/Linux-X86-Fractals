Some tips

How to install OpenGL/Glut:

sudo apt-get install aptitude

sudo aptitude install freeglut3-dev

How to compile:

gcc gfxbrot.c -o gfxbrot -lGL -lGLU -lglut

How to run:

From terminal ./gfxbrot
